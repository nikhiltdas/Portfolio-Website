## 📂 Featured Projects

### Smart Trekking Navigation Device

An offline navigation system designed for trekkers using LoRa communication, GPS tracking, mesh networking, SOS alerts, and environmental hazard detection.

### IoT Energy Monitoring System

A smart power monitoring solution using ESP32, SCT-013 current sensor, and ZMPT101B voltage sensor to measure voltage, current, and power consumption in real time.

### Smart Waste Segregation System

An automated waste classification system that helps separate different categories of waste using sensors and intelligent decision-making.

### School AI Assistant

A Raspberry Pi-based AI assistant developed to provide information and assistance within a school environment.
